## Sobre

Sou desenvolvedora Full Stack com foco em aplicações web, automação e análise de dados. Este portfólio reúne os projetos que desenvolvi durante minha formação em ADS e pelo programa Oracle Next Education.

---

## Projetos em Destaque

### Bela Venda — Sistema de Gestão
Sistema completo para revendedoras de cosméticos com controle de vendas, estoque e financeiro. Inclui automação de leitura de nota fiscal via PDF e controle inteligente de pedidos.

`React` `Supabase` `PostgreSQL` `PDF Parsing`

---

### Monitor Regulatório
Sistema de automação que coleta e organiza dados de portais financeiros (BACEN, CVM) via web scraping. Elimina verificação manual e mantém as informações atualizadas continuamente.

`.NET` `C#` `Web Scraping` `ETL`

---

### Sentiment API
API de análise de sentimentos que utiliza NLP e machine learning para classificar textos automaticamente. Suporta múltiplos idiomas e análise escalável de dados textuais.

`Python` `FastAPI` `NLP` `Machine Learning`
